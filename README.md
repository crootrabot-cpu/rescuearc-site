# RescueArc Concept Site

Static concept website for a throwable inflatable rescue device currently using the working title **RescueArc**.

## What this is
- shareable invention-story website
- commercialization roadmap surface
- schematic and concept-proof artifact

## What this is not
- not certified lifesaving equipment
- not a preorder flow
- not proof that engineering, compliance, or patent risk is solved

## Pages
- `index.html` — main concept site
- `assets/lifeguard-scene.svg` — hero/editorial scenario art
- `assets/deployment-sequence.svg` — deployment diagram
- `assets/prototype-architecture.svg` — cutaway concept schematic
- `docs/superpowers/specs/2026-06-28-rescuearc-concept-design.md` — design/spec
- `docs/plans/2026-06-28-rescuearc-site-implementation-plan.md` — implementation plan
- `docs/research/2026-06-28-rescuearc-prior-art-and-roadmap.md` — research and business roadmap note
